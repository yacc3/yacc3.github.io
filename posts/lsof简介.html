list open files 查看当前打开文件

文件

  • 普通文件
  • 目录
  • 网络文件系统的文件
  • 字符或设备文件
  • (函数)共享库
  • 管道，命名管道
  • 符号链接
  • 网络文件（例如：NFS file、网络socket，unix域名socket）
  • 还有其它类型的文件，等等

command

  • -a 列出打开文件存在的进程
  • -c 列出指定进程所打开的文件进程名>
  • -g 列出GID号进程详情
  • -d 列出占用该文件号的进程文件号>
  • +d 列出目录下被打开的文件目录>
  • +D 递归列出目录下被打开的文件目录>
  • -n 列出使用NFS的文件目录>
  • -i 列出符合条件的进程。（4、6、协议、:端口、 @ip ）条件>
  • -p 列出指定进程号所打开的文件进程号>
  • -u 列出UID号进程详情
  • -h 显示帮助信息
  • -v 显示版本信息

举例

  • lsof /bin/bash
  • lsof -u C
  • lsof -c mysql
  • lsof -u C -c mysql
  • lsof -p 11968
  • lsof -i:8080
  • lsof -i tcp
  • lsof -a -u C -i
